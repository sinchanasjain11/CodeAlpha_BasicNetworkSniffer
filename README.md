# Basic Network Sniffer 🔐

## About
A Basic Network Sniffer built using Python 
during CodeAlpha Cybersecurity Internship.
This tool captures live network packets and 
displays useful information.

## Features
- Captures live network packets
- Displays Source IP address
- Displays Destination IP address
- Shows Protocol type (TCP/UDP/ICMP)
- Shows Port numbers
- Displays timestamp of each packet

## Tools and Libraries Used
- Python 3
- Scapy Library
- Npcap

## Requirements
- Python 3.x
- Scapy
- Npcap (for Windows)

## Installation
Step 1: Install Python from python.org
Step 2: Install Scapy
        pip install scapy
Step 3: Install Npcap from npcap.com

## How to Run
Step 1: Open CMD as Administrator
Step 2: Go to file location
        cd C:\Users\admin\Desktop
Step 3: Run the program
        python sniffer.py
